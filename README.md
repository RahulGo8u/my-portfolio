# Rahul Anand — Portfolio

Personal portfolio website built with **Next.js 15** and **Tailwind CSS**, deployed on Vercel.

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Deployment**: Vercel

## Sections

- Hero with stats
- About with quick facts
- Technical skills matrix (incl. AI/Bedrock)
- Experience timeline
- Featured projects
- Certifications & Education
- Contact

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Deploy to Vercel

1. Push to GitHub
2. Import repo in [vercel.com/new](https://vercel.com/new)
3. Vercel auto-detects Next.js — click **Deploy**
